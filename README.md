PROGRAM:
```python
#Program to find the distance between two points.
#Developed by: SWETHA P
#RegisterNumber:22008542
import math
x2=[10,6]
x1=[4,2]
dist=math.sqrt(((x2[0]-x1[0])**2)+((x2[1]-x1[1])**2))
print("{:.2f}".format(dist))
```
